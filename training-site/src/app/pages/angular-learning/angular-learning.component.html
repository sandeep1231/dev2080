Common Errors & Quick Fixes

❌ Template Parse Error

Cause: Missing import of a component or directive

Fix: Import the component/directive in your component's imports array or in a shared module.

❌ NullInjectorError

Cause: Service not registered/provided

Fix: Provide the service in the component's providers array or in a parent injector.

❌ Cannot Bind to 'ngModel'

Cause: FormsModule not imported

Fix: Import FormsModule in your component or module for two-way binding.

❌ Change Detection Issues

Cause: Angular not detecting external changes

Fix: Use ngZone or markForCheck() to trigger change detection manually.
