Core Concepts at a Glance

Components & Templates

Components are building blocks with UI logic. Templates define markup using Angular syntax for dynamic rendering. Standalone components simplify architecture.

Data Binding

Keep view and component in sync: interpolation, property binding, event binding, and two-way binding with ngModel.

Signals & Reactivity

Angular's new reactivity model: signals, computed values, and effects for fine-grained state management without RxJS complexity.

Routing & Navigation

Map URLs to components, pass route params, implement guards, resolvers, and lazy loading for performance.

Dependency Injection

Angular's powerful DI system: injectable services, provider hierarchies, inject() function, and injection tokens.

Forms & Validation

Template-driven and reactive forms with built-in validators, custom validators, and async validation patterns.

HTTP & API Integration

HttpClient for REST APIs, interceptors for auth tokens, error handling, caching strategies, and retry logic.

Testing with Vitest

Common Errors & Quick Fixes

Template Parse Error

Cause: Missing import of a component or directive

Fix: Import the component/directive in your component's imports array (standalone) or in a shared module.

NullInjectorError

Cause: Service not registered/provided

Fix: Add @Injectable({ providedIn: 'root' }) or provide the service in the component's providers array.

Cannot Bind to 'ngModel'

Cause: FormsModule not imported

Fix: Import FormsModule in your component's imports array for two-way binding with [(ngModel)].

ExpressionChangedAfterItHasBeenCheckedError

Cause: Value changed during change detection cycle

Fix: Use signals instead of manual state changes, or wrap in setTimeout/afterNextRender.

Circular Dependency Detected

Cause: Two modules/services importing each other

Fix: Refactor shared logic into a third service or use injection tokens to break the cycle.

Zone.js Not Found / SSR Issues

Cause: Missing polyfill or incorrect SSR configuration

Fix: Use provideZonelessChangeDetection() for zoneless Angular or verify polyfills.ts configuration.
