Node.js Interview Questions (Beginner)

  • What is Node.js event loop? - Handles async callbacks non‑blocking.
  • What is Express middleware? - Functions that can modify request/response.
  • How do you create a route? - app.get('/path', handler).
  • What is REST? - Architectural style for stateless APIs using HTTP verbs.
  • How to manage env config? - process.env, dotenv, config libraries.
  • How to connect MongoDB? - Using official driver or Mongoose.
  • What are status codes? - 2xx success, 4xx client errors, 5xx server errors.
  • What is package.json? - Project metadata and scripts/dependencies.
