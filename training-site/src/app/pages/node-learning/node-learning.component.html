Common Errors & Quick Fixes

Cannot Find Module

Cause: Package not installed or wrong import path

Fix: Run npm install package-name or verify the import path and file extension.

EADDRINUSE: Port Already in Use

Cause: Another process is using the port

Fix: Kill the existing process or use a different port via environment variable.

CORS Policy Error

Cause: Cross-origin request blocked by browser

Fix: Install and configure the cors middleware with proper origin whitelist.

UnhandledPromiseRejection

Cause: Async error not caught

Fix: Always use try/catch with async/await or .catch() with promises. Add global error handler.

ECONNREFUSED

Cause: Database or external service not running

Fix: Verify connection string, check if MongoDB/Redis is running and IP is whitelisted.

ERR_HTTP_HEADERS_SENT

Cause: Sending response after headers already sent

Fix: Add return after res.send/json/redirect to prevent further execution.
