Common Errors & Quick Fixes

Connection Failed

Cause: Invalid connection string or MongoDB not running

Fix: Verify connection string, check IP whitelist in Atlas, ensure service is running.

Authentication Failed

Cause: Wrong credentials or user not created

Fix: Verify username/password in Atlas, create database user with proper roles.

Duplicate Key Error (E11000)

Cause: Unique index constraint violation

Fix: Ensure field values are unique, handle errors gracefully, or remove unique constraint.

Document Too Large (16MB limit)

Cause: Storing too much data in a single document

Fix: Use GridFS for large files, or restructure schema with references instead of embedding.

Slow Query Performance

Cause: Missing indexes or inefficient query patterns

Fix: Use explain() to analyze, create indexes on queried fields, use projection to limit fields.

Mongoose Validation Error

Cause: Data doesn't match Mongoose schema definition

Fix: Validate data before saving, handle ValidationError in catch block, check required fields.
