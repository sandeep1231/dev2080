MongoDB Training & Learning Hub

Master NoSQL data modeling, CRUD operations, and performance tuning with MongoDB and Mongoose.

Why MongoDB?

  • Flexible schema for evolving product needs
  • Rich JSON-like documents for nested data
  • Horizontal scaling and high availability

Core Concepts

  • Documents, Collections, Databases
  • CRUD: insertOne, find, updateOne, deleteMany
  • Indexes (single, compound, text) and performance
  • Aggregation framework (match, group, project)
  • Mongoose ODM: Schemas, Models, Validation, Middleware

Common Errors & Fixes

  • ECONNREFUSED: Ensure MongoDB server is running and connection string is correct.
  • CastError in Mongoose: Validate ObjectId params before querying.
  • Performance issues: Add appropriate indexes; analyze with `explain()`.

Quick Start (Node.js + Mongoose)

npm install mongoose

// connect
import mongoose from 'mongoose';
await mongoose.connect('mongodb://127.0.0.1:27017/training');

// schema + model
const courseSchema = new mongoose.Schema({
  title: { type: String, required: true },
  hours: Number,
}, { timestamps: true });
const Course = mongoose.model('Course', courseSchema);

// CRUD
await Course.create({ title: 'MongoDB Essentials', hours: 6 });
const list = await Course.find().sort('-createdAt');
